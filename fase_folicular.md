# Fase folicular (7–10 días)

**Enfoque hormonal.** El hipotálamo envía señales a la hipófisis para que libere hormona foliculoestimulante (FSH) hacia los ovarios, indicándoles que se preparen para liberar un óvulo. Varios folículos comienzan a desarrollarse. El estrógeno aumenta para engrosar el revestimiento uterino.

**Enfoque corporal.** La energía física aumenta gradualmente; al inicio hay poca o ninguna secreción vaginal, luego se incrementa (amarilla o blanca, pegajosa).

**Enfoque de estilo de vida.** Ideal para proyectos nuevos y creativos, trabajo intelectual desafiante y actividades sociales. Etapa de mayor apertura a nuevas experiencias.

**Enfoque alimentario.** Alimentos frescos y ligeros: ensaladas prensadas (kimchi, chucrut), vegetales, proteínas magras, legumbres y semillas germinadas, granos energéticos. Métodos de cocción ligeros como vapor o salteado.

**Enfoque de ejercicio.** Probar actividades nuevas (Zumba, yoga sculpt, etc.). Mayor facilidad para aprender y consolidar nuevos hábitos físicos.
